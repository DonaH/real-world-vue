# real-world-vue

## Project setup
```
yarn install

```

### Compiles and hot-reloads for development
##### http://localhost:8080
```
yarn serve
```

### Compiles and minifies for production
##### /dist folder
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### View or install dependencies
##### http://localhost:8000
```
vue ui
```

### Deployment via Render
[Render Cloud Deployment](https://render.com/)
